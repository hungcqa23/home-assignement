Take-home Assignment
Junior Full-stack Engineer 🚀

Photo upload and comment
Overall goal:
Create a Next.js web application allowing users to upload photos and comment on them.
You can decide the app’s UI. It can be very simple.
The goal is to develop a functional application, covering both front and back end. There are no strict guidelines on implementation details.
The purpose of this assignment is to assess your coding skills and tech stack knowledge.
This is expected to be completed within a day. However, please feel free to provide an estimated timeline based on your current knowledge and experience.

Communications:
Please “Make a copy” of this document, add your name to the title and share it with our Full Stack Engineer (tho.nguyen@qode.world), and our HR Team (lucy@qode.world).

If you have any questions, you can add comments directly on the document and mention them.

Tech stack:
Frontend: Next.js, TypeScript or Javascript, https://ant.design/
Backend: Node.js or Next JS API routes, prisma ORM
Database: PostgreSQL

Deliverables:
Source Code: Provide a link to your GitHub repository.
Executable: Deploy the application and provide the link for the live version.

Function:
Upload a photo
Add a comment to a photo
Display all uploaded photos and its comments

Notes:
The images don't need to be persistent. The app only needs to demo the upload functionality.
If you couldn’t find a way to host a server for the user to upload the pictures, provide clear instructions for running the application.
